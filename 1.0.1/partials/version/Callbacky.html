Callbacky

Description

Callbacky is a Javascript event handler that allows multiple parts of an application communicate with each other

You have to call init before all other methods.

Methods

  • bind(key, method)

    Bind a callback to an event, associated to a string.
    When you call trigger method with same string, the callback will be called.
    You can associate two or more callbacks on a event

    Parameters
    ParamTypeDetails
    keystring

    key to identify the event

    methodmethod

    callback will be called when trigger same key

    Example

    Callbacky.bind('eventName', function(args){ 
    console.log(args)) 
};

Callbacky.trigger('eventName', 'world');
// IT EXECUTES: console.log('world');

  • clean(key)

    Clean all callbacks associated to an event

    Parameters
    ParamTypeDetails
    keystring

    key to identify the event

    Example

    Callbacky.bind('eventName', function(args){ 
    console.log(args)) 
};

Callbacky.clean('eventName');

Callbacky.trigger('eventName', 'world');
// It does nothing

  • cleanAll()

    Clean all callbacks associated to all events

    Example

    Callbacky.bind('eventName', function(args){ 
    console.log(args)) 
};

Callbacky.cleanAll();

Callbacky.trigger('eventName', 'world');
// It does nothing

  • init(options)

    This method is used to initialize the Callbacky module.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    loggerLogger

    any object containing the following methods: debug, log, info, warn, error

    (default: {debug:function(){},log:function(){},info:function(){},warn:function(){},error:function(){}})

    Example

    Callbacky.init({
    logger: console
});

  • trigger(key, arguments)

    Trigger an event, binded before with bind method, and call the callbacks associated to event

    Parameters
    ParamTypeDetails
    keystring

    key to identify the event

    argumentsObject

    arguments to pass to associated callbacks

    (default: undefined)

    Example

    Callbacky.bind('eventName', function(args){ 
    console.log(args)) 
};

Callbacky.trigger('eventName', 'world');
// IT EXECUTES: console.log('world');
